# Analyse Complète dApp xArtists

## Architecture Globale

- **Frontend** : React 18 + Vite + TypeScript, Tailwind, React Router, MultiversX SDK-dapp v5
- **Smart Contracts** : Rust (no_std) pour nft-staking, tro-staking, btc-bridge sur MultiversX
- **AI Layer** : LIA v5 Agent avec UniversalExecutor pour actions DeFi autonomes (xExchange swaps, staking)
- **Backend/Services** : On-chain queries, price feeds, wallet integration
- **Deployment** : GitHub Pages (gh-pages), contracts sur devnet/mainnet

## Fonctionnalités Clés
- Dashboard principal avec live stats EGLD, TRO, pools
- Demos : Faucet, Escrow, Tagrity (auth art physique), Burnify
- Staking NFT/TRO avec rewards
- Bridge Bitcoin L2
- AI-driven liquidity & trading suggestions
