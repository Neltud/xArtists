## 🌟 Présentation

xArtists est une **DApp complète** dédiée aux artistes, collectionneurs et créateurs sur l'écosystème MultiversX.

**Live Demo** : [https://neltud.github.io/xArtists](https://neltud.github.io/xArtists)

## ✨ Fonctionnalités Principales (v1.0.0)

- 🎨 **Minting NFT & RWA** (art physique avec escrow sécurisé)
- 🔒 **Staking hybride** NFT + TRO token
- 🧠 **Dashboard LIA v5** avec analyse IA des œuvres
- 🔑 **Intégration xPortal complète** (In-App Browser + xGenie AI)
- 🛡️ **Monitoring on-chain** en temps réel
- 📊 **Governance DAO**, Marketplace, On-ramp Moonpay
- 🤖 Bots Discord & X

## 🛠 Stack Technique

- Frontend: React 18 + Vite + TypeScript + Tailwind + @multiversx/sdk-dapp v5
- Contracts: Rust WASM (MultiversX)
- Wallet: xPortal, WalletConnect v2, NativeAuth
- IA: LIA Agents + xGenie
- Monitoring & Sécurité: CodeQL, Dependabot, on-chain alerts

## 🚀 Quick Start

```bash
git clone https://github.com/Neltud/xArtists.git
cd xArtists
pnpm install
pnpm dev
```

Consultez `MONITORING.md`, `XPORTAL_INTEGRATION.md` et `DEPLOY.md`.
